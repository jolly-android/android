# Kotlin Coroutines Demo App

A comprehensive Android application demonstrating 44 different Kotlin Coroutines examples across 4 categories.

## Quick Start

1. Open project in Android Studio
2. Sync Gradle files
3. Run the app on an emulator or device
4. Explore the 4 tabs with interactive demos

## What's Inside

- **Basic Coroutines** (8 demos) - launch, async, cancellation, dispatchers, etc.
- **Flow** (12 demos) - StateFlow, SharedFlow, operators, transformations
- **Channels** (12 demos) - buffering, fan-out/in, pipelines, producers
- **Real World** (12 demos) - API calls, caching, retry, pagination, etc.

## Key Features

✅ 44 Interactive demos  
✅ Real-time logging  
✅ Visual feedback with loading states  
✅ Material Design 3 UI  
✅ Clean architecture with ViewModels  
✅ Comprehensive comments and explanations
